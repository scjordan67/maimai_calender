# 每日万年历插件

适配 [MaiBot / 麦麦](https://docs.mai-mai.org) 的定时万年历插件，每天在指定群聊或私聊推送今日完整历法信息，底部附一条随机古典文学语录。

## 功能说明

| 内容 | 实现方式 |
|------|----------|
| 农历日期、干支纪年、生肖 | [`lunardate`](https://pypi.org/project/lunardate/) 库精确转换 |
| 节气 | 寿星万年历公式（21 世纪精度 ±1 天） |
| 宜 / 忌 | 基于当日干支的确定性算法（同一天结果固定） |
| 今日吉时 | 日天干推时法（传统历法简化版） |
| 文学语录 | 从 `data/quotes.json` 随机抽取 |

语录库共收录 **131 条**，涵盖：四书五经、先秦诸子、楚辞、唐诗、宋词、史记、明清小说等。

## 安装

### 1. 复制插件目录

将整个 `daily_calendar/` 文件夹放到 MaiBot 的 `plugins/` 目录下：

```
MaiBot/
└── plugins/
    └── daily_calendar/   ← 放在这里
```

### 2. 安装 Python 依赖

```bash
pip install lunardate
```

> `lunardate` 是唯一的额外依赖，用于农历日期转换。节气、宜忌、吉时均为纯 Python 实现，无需其他库。

### 3. 启动 Bot

首次启动后，系统会在 `plugins/daily_calendar/` 下自动生成 `config.toml`，编辑该文件完成配置。

## 配置说明

> `config.toml` 由麦麦自动生成，**请勿手动创建**，直接编辑生成后的文件即可。

```toml
[plugin]
enabled = true              # 是否启用插件

[schedule]
send_time = "08:00"         # 每日推送时间，格式 HH:MM
                            # 例：07:30 表示每天早上七点半

[targets]
group_ids = []              # 要推送的群号列表（字符串格式）
                            # 例：["123456789", "987654321"]
user_ids  = []              # 要私聊推送的用户 ID 列表（字符串格式）
                            # 例：["10001", "10002"]
platform  = "qq"            # 平台标识，目前仅支持 qq
```

### 配置示例

每天早上 8 点向两个群和一个私聊用户推送：

```toml
[plugin]
enabled = true

[schedule]
send_time = "08:00"

[targets]
group_ids = ["114514191", "998244353"]
user_ids  = ["10001"]
platform  = "qq"
```

## 扩充语录库

语录保存在 `data/quotes.json`，格式如下：

```json
{
  "quotes": [
    {
      "text": "天行健，君子以自强不息。",
      "source": "《周易·乾卦》",
      "author": ""
    },
    {
      "text": "路漫漫其修远兮，吾将上下而求索。",
      "source": "《离骚》",
      "author": "屈原"
    }
  ]
}
```

字段说明：
- `text`：语录正文（必填）
- `source`：出处，如《论语·学而》（必填）
- `author`：作者，不明或集体创作可留空字符串（必填，可为 `""`）

## 技术细节

### 节气计算（寿星公式）

```
节气日 = int(Y × 0.2422 + C) - int((Y - 1) / 4)
```

其中 Y 为年份后两位，C 为各节气常数（见代码中 `SOLAR_TERMS_INFO`）。适用于 21 世纪，精度 ±1 天。

### 干支推算

- **年柱**：以农历年份为准，使用 `lunardate` 获取，基准年 1924 年（甲子年）
- **月柱**：五虎遁年法，由年天干推算寅月起始天干
- **日柱**：以 2000 年 1 月 7 日（甲子日）为基准，按天偏移量计算

### 宜忌算法

以当日干支字符串为随机种子，从宜忌活动池中确定性抽取，保证同一天结果一致且不重叠。

### 吉时算法

按日天干查表，每日固定对应四个吉时（传统历法简化版）：

| 日天干 | 吉时 |
|--------|------|
| 甲 | 寅、辰、申、戌 |
| 乙 | 丑、卯、未、酉 |
| 丙 | 子、辰、午、戌 |
| 丁 | 丑、卯、巳、亥 |
| 戊 | 寅、午、申、戌 |
| 己 | 丑、巳、未、亥 |
| 庚 | 子、寅、申、戌 |
| 辛 | 卯、巳、未、酉 |
| 壬 | 子、辰、午、戌 |
| 癸 | 丑、卯、巳、酉 |

### 定时器实现

使用纯 asyncio（无额外依赖），Bot 启动后延迟 2 秒自动激活。每次任务执行后计算下一个触发时刻，精度为分钟级。

## 常见问题

**Q：农历显示为"（需安装 lunardate 库）"？**  
A：执行 `pip install lunardate` 后重启 Bot。

**Q：如何修改推送时间？**  
A：编辑 `config.toml` 中的 `send_time` 字段后重启 Bot。

**Q：定时任务没有触发？**  
A：检查 `targets.group_ids` 或 `targets.user_ids` 是否已正确填写；确认 Bot 已加入对应群组。

**Q：节气日期偶尔差一天？**  
A：寿星公式存在 ±1 天误差，属正常现象。
